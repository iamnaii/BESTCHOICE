# Post 3 — TikTok Tutorial ขั้นตอนผ่อน (Education Pillar)

**Platform:** TikTok
**วันที่โพสต์:** พฤหัสบดี 3 เมษายน 2026
**Format:** Tutorial / How-to
**ความยาว:** 45-60 วินาที
**Pillar:** วิธีผ่อน (Education)
**CTA:** LINE OA

---

## Script / Storyboard

### Hook (0-3 วินาที) — ต้องดึงดูดทันที
**ภาพ:** หน้าจอ text บน background เรียบง่าย หรือ staff มองกล้องตรงๆ
**Text overlay:** `"ผ่อน iPhone กับ BESTCHOICE ง่ายมาก — 3 ขั้นตอนเท่านั้น 👆"`
**เสียง:** เสียงพูดตรงๆ หรือเพลง upbeat เบาๆ

---

### Scene 1 (3-20 วินาที) — ขั้นตอนที่ 1
**ภาพ:** มือถือบัตรประชาชน close-up
**Text overlay:**
```
ขั้นตอนที่ 1 🪪
ยื่นบัตรประชาชน
ใบเดียวเท่านั้น!
```
**Voiceover:** "ไม่ต้องมีบัตรเครดิต ไม่ต้องมีคนค้ำ ไม่เช็คเครดิตบูโร — บัตรประชาชนใบเดียว พอแล้ว"

---

### Scene 2 (20-35 วินาที) — ขั้นตอนที่ 2
**ภาพ:** นาฬิกา หรือ animation เวลา 15 นาที
**Text overlay:**
```
ขั้นตอนที่ 2 ⏱️
รอผลอนุมัติ
แค่ 15 นาที
```
**Voiceover:** "ยื่นแล้วรอนิดเดียว — 15 นาที รู้ผลเลย ไม่ต้องรอข้ามวัน"

---

### Scene 3 (35-50 วินาที) — ขั้นตอนที่ 3
**ภาพ:** Staff ส่งมอบ iPhone ให้ลูกค้า ยิ้มแย้ม
**Text overlay:**
```
ขั้นตอนที่ 3 📱
รับเครื่องได้เลย!
ผ่อนเดือนละ XXX บาท*
```
**Voiceover:** "อนุมัติปุ๊บ รับเครื่องได้เลย ผ่อนเดือนละ [ราคา] บาท สบายๆ"

---

### Scene 4 (50-60 วินาที) — Summary + CTA
**ภาพ:** Text summary สวยงาม หรือ infographic 3 ขั้นตอน
**Text overlay:**
```
สรุป: 3 ขั้นตอนเท่านั้น!
1️⃣ ยื่นบัตรปชช
2️⃣ รอ 15 นาที
3️⃣ รับ iPhone ใหม่ 🎉

ทักมาเลย LINE OA: @bestchoice
```

## Visual Notes

- สไตล์: clean, minimal — text อ่านง่ายบนมือถือ
- สีแบรนด์ BESTCHOICE บน text overlay
- แต่ละขั้นตอนใช้ transition สะอาด (ไม่ overdone)
- ถ้ามี infographic สวยๆ ประกอบ 3 ขั้นตอน ยิ่งดี
- *ราคาผ่อน (XXX) ต้องได้รับการยืนยันจากฝ่ายขายก่อนถ่ายทำ*
