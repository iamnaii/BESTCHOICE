# Walk-in Sales SOP — BESTCHOICE ลพบุรี

**เวอร์ชัน**: 1.0
**อัปเดต**: 2026-04-02
**เจ้าของ**: CSO — BESTCHOICE

---

## ภาพรวม

SOP นี้คือขั้นตอนมาตรฐานสำหรับพนักงานหน้าร้านในการรับลูกค้า Walk-in
ตั้งแต่ลูกค้าเดินเข้าร้าน → ปิดการขาย → รับสัญญา

**สินค้าที่จำหน่าย:**
- iPhone มือ1 (ใหม่) — ทุกรุ่น
- iPhone มือ2 (มือสอง) — คัดสภาพดี
- iPad มือ1 (ใหม่) — ทุกรุ่น

**ห้ามเสนอ:** Android, iPad มือ2, อุปกรณ์อื่นที่ไม่อยู่ใน portfolio

---

## จุดขายหลัก (ท่องให้ขึ้นใจ)

1. **ผ่อนได้ด้วยบัตรประชาชนใบเดียว**
2. **ไม่เช็คเครดิตบูโร**
3. **ทุกอาชีพผ่อนได้**
4. **ดาวน์ถูก + ค่างวดต่ำที่สุด**
5. **อนุมัติภายใน 15 นาที**

---

## ขั้นตอนการขาย (7 Steps)

---

### Step 1: ต้อนรับ (Greet) — 0-2 นาที

**เป้าหมาย:** ทำให้ลูกค้ารู้สึกสบายใจ ไม่ถูก pressure

**Script ต้อนรับ:**
```
สวัสดีครับ/ค่ะ ยินดีต้อนรับสู่ BESTCHOICE นะครับ/ค่ะ

วันนี้มาเพื่อดูอะไรเป็นพิเศษไหมครับ/ค่ะ?
หรือให้ช่วยแนะนำก็ได้ครับ/ค่ะ
```

**Do's:**
- ยิ้มและทักทายทุกคนที่เข้าร้าน
- ให้พื้นที่ลูกค้าดูสินค้าก่อน ไม่ตาม
- เข้าหาเมื่อลูกค้าแสดงสัญญาณสนใจ (จับสินค้า, มองนาน)

**Don'ts:**
- อย่าพูดว่า "ซื้ออะไรดีครับ/ค่ะ?" ตั้งแต่แรก
- อย่าเดินตามใกล้เกินไป
- อย่าขัดจังหวะลูกค้าที่กำลังดูสินค้าอยู่

---

### Step 2: สำรวจความต้องการ (Discover) — 3-5 นาที

**เป้าหมาย:** เข้าใจว่าลูกค้าต้องการอะไร และมีงบเท่าไร

**Script สอบถาม:**
```
ขอสอบถามนิดนึงนะครับ/ค่ะ เพื่อจะแนะนำได้ตรงๆ เลย

มาดู iPhone หรือ iPad ครับ/ค่ะ?

แล้วงบประมาณคร่าวๆ มีประมาณเท่าไหร่ครับ/ค่ะ?
(ดาวน์และค่างวดรายเดือน)

มือหนึ่งหรือมือสองก็ได้ไหมครับ/ค่ะ?
```

**คำถามเพิ่มเติมตามสถานการณ์:**
- "ใช้เพื่ออะไรเป็นหลักครับ/ค่ะ? ถ่ายรูป เล่นเกม หรือทำงาน?"
- "ตอนนี้ใช้รุ่นอะไรอยู่ครับ/ค่ะ?"
- "เคยผ่อน iPhone มาก่อนไหมครับ/ค่ะ?"

**Note:** จดข้อมูลไว้ในระบบหรือกระดาษสำหรับส่งต่อทีม

---

### Step 3: แนะนำสินค้า (Recommend) — 5-10 นาที

**เป้าหมาย:** เสนอตัวเลือกที่เหมาะสม 2-3 รุ่น ไม่มากเกินไป

**หลักการเลือกรุ่นแนะนำ:**
- งบดาวน์น้อย → แนะนำ iPhone มือ2 หรือ iPhone รุ่นเก่า
- งบดาวน์ปานกลาง → iPhone 14 หรือ iPhone 15
- งบดาวน์สูง / อยากได้ใหม่ → iPhone 15 Pro หรือ iPhone 16

**Script แนะนำสินค้า:**
```
ตามที่บอกมา ผมแนะนำ [รุ่น A] กับ [รุ่น B] ครับ/ค่ะ

[รุ่น A] — เหมาะกว่าเพราะ [เหตุผล]
  ดาวน์ [X] บาท ผ่อน [Y] เดือน งวดละ [Z] บาท

[รุ่น B] — ถ้าอยากประหยัดกว่า หรือได้รุ่นใหม่กว่า
  ดาวน์ [X] บาท ผ่อน [Y] เดือน งวดละ [Z] บาท

ลองจับดูก่อนได้เลยนะครับ/ค่ะ
```

**Tips:**
- ให้ลูกค้าจับสินค้าจริง
- เปิดหน้าจอให้ดู
- เปรียบเทียบรุ่นไม่เกิน 3 รุ่นในครั้งเดียว

---

### Step 4: อธิบายเงื่อนไขผ่อน (Explain Terms) — 5-7 นาที

**เป้าหมาย:** ลูกค้าเข้าใจเงื่อนไขชัดเจน ไม่มีข้อสงสัยเรื่องค่าใช้จ่าย

**Script อธิบายเงื่อนไข:**
```
เรื่องการผ่อน ขอเล่าให้ฟังนะครับ/ค่ะ

BESTCHOICE ผ่อนง่ายมาก:
• ใช้บัตรประชาชนใบเดียว ไม่ต้องมีบัตรเครดิต
• ไม่เช็คเครดิตบูโร
• ทุกอาชีพผ่อนได้เลย

เอกสารที่ต้องใช้:
1. บัตรประชาชน (ถ่ายรูปหน้า-หลัง)
2. สลิปเงินเดือนหรือหลักฐานรายได้ 2-3 เดือน

อนุมัติเร็วมาก ประมาณ 15 นาที รู้ผลทันที

สำหรับ [รุ่นที่สนใจ]:
  ดาวน์ [X] บาท
  ผ่อน [Y] เดือน งวดละ [Z] บาท
  ชำระผ่าน Promptpay หรือโอนธนาคารรายเดือน
```

**ต้องพูดชัดเจนเสมอ:**
- จำนวนดาวน์ที่ต้องจ่ายวันนี้
- ค่างวดรายเดือน
- จำนวนงวดทั้งหมด
- วิธีชำระงวด

---

### Step 5: จัดการ Objection (Handle) — ตามความจำเป็น

**เป้าหมาย:** ตอบข้อกังวลของลูกค้าอย่างตรงไปตรงมา ไม่กดดัน

#### กรณี: "ค่างวดแพงไป"
```
เข้าใจเลยครับ/ค่ะ

ลองปรับเป็นผ่อน 18 หรือ 24 เดือนได้นะครับ/ค่ะ
ค่างวดจะเบาลงมาก

เช่น [รุ่น] ผ่อน 24 เดือน งวดละแค่ [X] บาทเอง
หรือเปลี่ยนเป็นรุ่นที่ราคาถูกกว่าก็ได้ครับ/ค่ะ
```

#### กรณี: "กลัวไม่ผ่าน / ประวัติไม่ดี"
```
ไม่ต้องกังวลเลยครับ/ค่ะ
เราไม่เช็คเครดิตบูโร

สิ่งที่ดูคือรายได้และความสามารถผ่อนจริงๆ
ถ้ามีหลักฐานรายได้ โอกาสผ่านสูงมากครับ/ค่ะ

ลองยื่นดูก่อนได้เลย ไม่มีค่าใช้จ่ายในการยื่น
```

#### กรณี: "อยากได้ Android / Samsung"
```
ขอบคุณที่บอกนะครับ/ค่ะ

BESTCHOICE เชี่ยวชาญเฉพาะ iPhone และ iPad
Android เราไม่มีในสต็อกครับ/ค่ะ

แต่ถ้ายังเปิดรับ iPhone อยู่ เราช่วยหา iPhone
ในงบเดียวกันให้ได้ครับ/ค่ะ สนใจให้แนะนำไหมครับ/ค่ะ?
```

#### กรณี: "ขอคิดดูก่อน"
```
ได้เลยครับ/ค่ะ ไม่รีบนะ

ขอเบอร์ไว้ส่งข้อมูลเพิ่มเติมให้ได้ไหมครับ/ค่ะ?
จะส่งตารางผ่อนของรุ่นที่สนใจไปให้ดูสบายๆ ที่บ้านได้เลย

ถ้าตัดสินใจแล้ว กลับมาได้เลยนะครับ/ค่ะ ยินดีเสมอ
```

#### กรณี: "เคยโดนปฏิเสธจากที่อื่น"
```
เสียใจด้วยนะครับ/ค่ะ

ที่เราต่างจากธนาคารคือเราไม่ได้ดูแค่ประวัติเครดิต
ดูรายได้จริงและความสม่ำเสมอของรายได้

ลองยื่นเอกสารดูก่อนได้เลยครับ/ค่ะ
ถ้ายื่นแล้วทราบผลใน 15 นาทีเลย
```

---

### Step 6: ยื่นเอกสาร & รอผล (Apply) — 15-20 นาที

**เป้าหมาย:** กระบวนการราบรื่น เร็ว ลูกค้าไม่รู้สึกรอนาน

**เมื่อลูกค้าตัดสินใจ:**

```
เยี่ยมเลยครับ/ค่ะ!

ขอเอกสารสักครู่นะครับ/ค่ะ:
1. บัตรประชาชน (ขอถ่ายรูปหน้า-หลัง)
2. สลิปเงินเดือนหรือหลักฐานรายได้

ใช้เวลาประมาณ 15 นาที รู้ผลเลยครับ/ค่ะ
ระหว่างรอ ชาหรือน้ำรับรองได้นะครับ/ค่ะ
```

**ระหว่างรอผลอนุมัติ:**
- นำเสนอ accessories / upsell (เคส, ฟิล์ม, ประกันภัย)
- แนะนำวิธีชำระงวดรายเดือน
- อธิบาย LINE OA สำหรับรับการแจ้งเตือน

**Script Upsell:**
```
ระหว่างรอผลอนุมัติ ขอแนะนำนิดนึงนะครับ/ค่ะ

เคสและฟิล์มกันรอยสำหรับ [รุ่น] มีให้เลือกเลยค่ะ
ราคาเริ่ม [X] บาท ติดให้ฟรีเลย ถ้าซื้อวันนี้

แล้วก็มีประกันภัยมือถือ [ราคา]/ปี ครอบคลุม:
• จอแตก / เสียหาย
• น้ำเข้า
สนใจรวมไปด้วยไหมครับ/ค่ะ?
```

---

### Step 7: ปิดการขาย & ส่งมอบ (Close) — 10-15 นาที

**เมื่ออนุมัติแล้ว:**

```
ยินดีด้วยนะครับ/ค่ะ อนุมัติแล้ว!

ขอดำเนินการต่อดังนี้:
1. เซ็นสัญญาผ่อนชำระ
2. จ่ายดาวน์ [X] บาท
3. รับเครื่อง + ใบเสร็จ

ชำระได้ทาง:
• Promptpay: [เลขบัญชี]
• โอนธนาคาร: [เลขบัญชี]
• เงินสด

หลังจากนี้ชำระงวดรายเดือนผ่าน [ช่องทาง]
งวดแรกวันที่ [วันที่ในสัญญา]
```

**ก่อนส่งมอบเครื่อง ตรวจสอบ:**
- [ ] ลูกค้าเซ็นสัญญาครบทุกหน้า
- [ ] รับดาวน์และออกใบเสร็จแล้ว
- [ ] กรอกข้อมูลในระบบ BESTCHOICE ครบ
- [ ] ลูกค้า Add LINE OA แล้ว (สำหรับแจ้งเตือนงวด)

**Script ส่งมอบ:**
```
ยินดีด้วยอีกครั้งนะครับ/ค่ะ!

นี่คือเครื่อง [รุ่น] ของคุณครับ/ค่ะ
ตรวจดูว่าครบไหม: [เครื่อง, กล่อง, สาย, ที่ชาร์จ]

ขอให้ใช้งานได้อย่างมีความสุขนะครับ/ค่ะ

สำคัญ: งวดแรกชำระวันที่ [วันที่] นะครับ/ค่ะ
ถ้ามีปัญหาอะไร ทักมาได้เลยที่ LINE: [LINE ID]

ขอบคุณที่เลือก BESTCHOICE นะครับ/ค่ะ
```

## กรณีพิเศษ

### ลูกค้าที่มากับคนที่มีอิทธิพลต่อการตัดสินใจ (เพื่อน, คู่สมรส)

```
ยินดีต้อนรับทุกท่านเลยครับ/ค่ะ
ดีเลยที่มาด้วยกัน จะได้ช่วยกันเลือกได้
```

- รวม "ที่ปรึกษา" เข้าในการสนทนา
- ตอบคำถามทุกคน ไม่ใช่แค่ผู้ซื้อ
- ขอมติร่วมก่อนปิดการขาย

### ลูกค้าขอต่อราคา / ขอส่วนลด

```
เรื่องราคาตัวเครื่อง ตรึงไว้ตามระบบครับ/ค่ะ
แต่ถ้าสนใจ เคส + ฟิล์มกันรอย เราให้ฟรีได้นะครับ/ค่ะ
```

- ห้ามลดราคาเครื่องโดยพลการ ต้องผ่านหัวหน้า
- สามารถเสนอ accessories ฟรีแทนได้

### ลูกค้าไม่ผ่านอนุมัติ

```
ขอโทษด้วยนะครับ/ค่ะ ครั้งนี้ยังผ่านไม่ได้

แต่ไม่ใช่ว่าจะผ่านไม่ได้ตลอดไปนะครับ/ค่ะ
ถ้าเพิ่มหลักฐานรายได้ได้ หรือผ่านไป 3-6 เดือน
สามารถลองสมัครใหม่ได้ครับ/ค่ะ

มีอะไรให้ช่วยเพิ่มเติมไหมครับ/ค่ะ?
```

- บันทึกข้อมูลลูกค้าในระบบ
- ระบุเหตุผลที่ไม่ผ่าน (เพื่อ CEO วิเคราะห์)

## Upsell & Cross-sell

### ลำดับการเสนอ Upsell
1. **เคส + ฟิล์ม** — เสนอก่อนเสมอ (อัตราสำเร็จสูงสุด)
2. **ประกันภัยมือถือ** — เสนอระหว่างรอผลอนุมัติ
3. **AirPods / Apple Watch** — เสนอหลังปิดการขายหลัก

### Script Upsell ตัวเลือกเร่งด่วน
```
ก่อนจะรับเครื่อง มี package เสริมให้พิจารณานะครับ/ค่ะ:

Package A — ป้องกันพื้นฐาน (XXX บาท)
• เคสกันกระแทก
• ฟิล์มกระจก 2 ชั้น
ติดให้ฟรีก่อนออกร้านเลย

Package B — คุ้มครองครบ (XXX บาท/ปี)
• ประกันภัยจอแตก + น้ำเข้า
• เคส + ฟิล์ม
สนใจ package ไหนครับ/ค่ะ?
```

## การบันทึกข้อมูลหลังการขาย

**ทุก Walk-in ต้องบันทึก:**

| ข้อมูล | บันทึกใน |
|--------|---------|
| ชื่อ-นามสกุล | ระบบ BESTCHOICE |
| รุ่นที่ซื้อ / สนใจ | ระบบ BESTCHOICE |
| ผลอนุมัติ | ระบบ BESTCHOICE |
| เหตุผลที่ไม่ซื้อ (ถ้ามี) | Form feedback |
| ช่องทางที่รู้จัก | ระบบ BESTCHOICE |

**Feedback ลูกค้าที่ไม่ซื้อ** — ส่งรายงานรายสัปดาห์ให้ CEO วิเคราะห์

## KPI สำหรับพนักงานหน้าร้าน

| KPI | เป้าหมาย | วัดอย่างไร |
|-----|----------|-----------|
| **Walk-in Conversion** | ≥ 30% | ลูกค้า Walk-in / สัญญาที่ปิดได้ |
| **Upsell Rate** | ≥ 40% | สัญญาที่มี upsell ≥ 1 รายการ |
| **CSAT** | ≥ 4.5/5 | จาก LINE OA review หลังรับเครื่อง |
| **Doc Completeness** | 100% | บันทึกครบทุก field ในระบบ |

## Do's & Don'ts สรุป

### Do's
- ยิ้มแย้มแจ่มใสทุกครั้ง
- ฟังก่อนพูด — เข้าใจความต้องการก่อนเสนอ
- บอกข้อมูลตรงไปตรงมา ไม่โอ้อวดเกินจริง
- เสนอตัวเลือกหลายแบบ ให้ลูกค้าเลือก
- บันทึกข้อมูลทุกราย ทั้งที่ซื้อและไม่ซื้อ

### Don'ts
- อย่า pressure หรือบีบบังคับ
- อย่าเสนอ Android หรือสินค้านอก portfolio
- อย่าให้ข้อมูลราคาหรือเงื่อนไขที่ไม่ตรงกับระบบ
- อย่าลดราคาโดยไม่ผ่านหัวหน้า
- อย่าปล่อยลูกค้ารอนานโดยไม่แจ้งความคืบหน้า

## ติดต่อสอบถาม / Escalation

- **ปัญหาอนุมัติ**: ติดต่อทีม Credit ทันที
