# LINE OA Deployment Plan — BESTCHOICE Sales Scripts

**เวอร์ชัน**: 1.0
**อัปเดต**: 2026-04-04
**เจ้าของ**: CSO — BESTCHOICE
**อ้างอิง Scripts**: `docs/sales-scripts/LINE_OA_SALES_SCRIPTS.md`

---

## ภาพรวม

แผนนี้ครอบคลุมการ deploy sales scripts ทั้งหมดเข้าสู่ LINE OA ของ BESTCHOICE ใน 3 ส่วน:

1. **Rich Menu** — Navigation UI หลักสำหรับลูกค้า
2. **Auto-reply & Keyword Triggers** — ตอบอัตโนมัติสำหรับ FAQ และนอกเวลาทำการ
3. **Quick Reply Buttons** — Shortcut ให้ลูกค้าเลือกตัวเลือกได้เร็ว

---

## ส่วนที่ 1: Rich Menu

### แนวคิด Rich Menu

Rich Menu คือ "แถบเมนู" ที่แสดงที่ด้านล่างหน้าต่าง chat — เป็น entry point แรกที่ลูกค้าเห็น

### Layout ที่แนะนำ (2 แถว × 3 ปุ่ม)

```
┌─────────────────────────────────────────┐
│  [📱 iPhone ใหม่] [📦 iPhone มือ2] [🖥 iPad] │
│  [💰 ราคา/ผ่อน]  [📋 เอกสาร]    [📞 ติดต่อ] │
└─────────────────────────────────────────┘
```

### รายละเอียดแต่ละปุ่ม

| ปุ่ม | Action Type | ค่า | เป้าหมาย |
|-----|------------|-----|---------|
| 📱 iPhone ใหม่ | Message | "สนใจ iPhone มือ1" | Trigger script แนะนำ iPhone ใหม่ |
| 📦 iPhone มือ2 | Message | "สนใจ iPhone มือ2" | Trigger script แนะนำ iPhone มือ2 |
| 🖥 iPad | Message | "สนใจ iPad" | Trigger script แนะนำ iPad |
| 💰 ราคา/ผ่อน | Message | "ถามเรื่องราคาและผ่อน" | แสดงตาราง FAQ ราคา |
| 📋 เอกสาร | Message | "ต้องใช้เอกสารอะไร" | แสดง FAQ เอกสาร |
| 📞 ติดต่อ | URI | `tel:[เบอร์ร้าน]` | โทรตรงหาร้าน |

### ขั้นตอนสร้าง Rich Menu

1. เข้า **LINE Official Account Manager** → **Chat** → **Rich menu**
2. กด **Create** → เลือก template 2×3
3. อัปโหลดรูป Rich Menu (ขนาด: 2500×1686px หรือ 1200×810px)
4. ตั้งค่า action แต่ละปุ่มตามตารางด้านบน
5. ตั้งชื่อ display ใต้ปุ่ม (ไม่เกิน 13 ตัวอักษร)
6. กด **Save** → **Publish**

> **หมายเหตุ**: Rich Menu แบบ free account รองรับ 1 menu ต่อ OA — ถ้าต้องการเปลี่ยน menu ตาม state ของลูกค้า ต้องใช้ LINE Messaging API (ต้องใช้ Developer account)

## ส่วนที่ 2: Auto-reply & Keyword Triggers

### 2.1 Welcome Message (Auto-reply เมื่อแอด OA ครั้งแรก)

ตั้งใน: **LINE OA Manager** → **Messaging** → **Greeting message**

```
สวัสดีค่ะ/ครับ ยินดีต้อนรับสู่ BESTCHOICE 🎉

ร้านผ่อน iPhone & iPad ลพบุรี
✅ ผ่อนได้ด้วยบัตรประชาชนใบเดียว
✅ ไม่เช็คเครดิตบูโร
✅ ทุกอาชีพผ่อนได้ — อนุมัติใน 15 นาที

👇 เลือกได้เลยจากเมนูด้านล่าง หรือทักมาหาเราได้เลยค่ะ/ครับ
```

---

### 2.2 Auto-reply นอกเวลาทำการ

ตั้งใน: **Messaging** → **Auto-reply** → **Time** (ตั้ง 19:00–09:00)

```
สวัสดีค่ะ/ครับ ขอบคุณที่ติดต่อ BESTCHOICE

ขณะนี้อยู่นอกเวลาทำการ (9:00–19:00 น. ทุกวัน)
เราจะตอบกลับให้เร็วที่สุดในเช้าวันทำการถัดไป

ถ้าเร่งด่วน โทร [เบอร์ติดต่อ] ได้เลยค่ะ/ครับ
ขอบคุณมากนะคะ/ครับ 🙏
```

---

### 2.3 Keyword Auto-reply

ตั้งใน: **Messaging** → **Auto-reply** → **Keyword**

| Keyword (ตั้งค่า) | Response |
|------------------|---------|
| `iPhone`, `ไอโฟน`, `i phone` | Script ขั้นตอน Qualify (ถามงบ + อาชีพ) พร้อม Quick Reply ปุ่มเลือกงบ |
| `iPad`, `ไอแพด` | "เรามี iPad มือ1 ใหม่ทุกรุ่น — สนใจรุ่นไหนคะ/ครับ? หรือให้แนะนำตามงบก็ได้เลย" |
| `ราคา`, `ผ่อน`, `งวด` | ตาราง FAQ ราคา + Quick Reply เลือกรุ่น |
| `เอกสาร`, `ต้องใช้อะไร` | Script เอกสาร (บัตรปชช. + สลิปเงินเดือน) |
| `ผ่านไหม`, `เครดิต`, `บูโร` | Script "ไม่เช็คเครดิตบูโร" |
| `ที่อยู่`, `ร้านอยู่ที่ไหน`, `แผนที่` | ที่อยู่ร้าน + Google Maps link |
| `ขอบคุณ`, `ok`, `โอเค` | "ยินดีเลยค่ะ/ครับ ถ้ามีคำถามอื่น ทักมาได้เสมอนะคะ/ครับ 😊" |

> **เคล็ดลับ**: ตั้ง keyword หลายคำสะกด (ภาษาไทย + ภาษาอังกฤษ) เพื่อ catch ทุก variation

### 2.4 Quick Reply Buttons

Quick Reply จะแสดงเป็นปุ่มให้ลูกค้ากดเลือก — ลด friction ในการพิมพ์

**ชุด 1: คัดกรองความสนใจ** (แนบกับ Welcome Message)
```
สนใจสินค้าประเภทไหน?
[📱 iPhone ใหม่] [📦 iPhone มือ2] [🖥 iPad ใหม่] [❓ ให้แนะนำ]
```

**ชุด 2: คัดกรองงบประมาณ** (แนบกับ script Qualify)
```
งบดาวน์ประมาณ?
[< 3,000 บาท] [3,000–6,000] [6,000–10,000] [10,000+]
```

**ชุด 3: ยืนยันความสนใจ** (หลังเสนอราคา)
```
[✅ สนใจ พร้อมสมัคร!] [🤔 ขอคิดดูก่อน] [🔄 ดูรุ่นอื่น]
```

**ชุด 4: เลือก CTA** (หลังสนใจ)
```
[🏠 มาที่ร้านเลย] [💻 สมัครออนไลน์] [📸 ส่งเอกสารทาง LINE]
```

## ส่วนที่ 3: Follow-up Sequences

ตั้งเป็น reminder ใน LINE OA หรือ CRM:

| วัน | ข้อความ | เงื่อนไข |
|----|---------|---------|
| วันที่ 1 | "ยังสนใจ [รุ่น] อยู่ไหมคะ/ครับ? มีคำถามเพิ่มเติมยินดีช่วยเลย" | ยังไม่สมัคร |
| วันที่ 3 | "[รุ่นที่สนใจ] ยังมีในสต็อกนะคะ/ครับ ถ้าสนใจทักมาได้เลย" | ยังไม่ตอบ |
| วันที่ 7 | "ทักถามรอบสุดท้ายนะคะ/ครับ ถ้าไม่สนใจแล้วก็ไม่เป็นไร ขอบคุณที่ให้โอกาสค่ะ/ครับ" | รอบสุดท้าย |

> **PDPA**: ส่งไม่เกิน 3 ครั้ง — หากไม่มีการตอบหลังครั้งที่ 3 ให้หยุด

## ส่วนที่ 4: Integration กับระบบ BESTCHOICE

### เชื่อม LINE LIFF

ระบบ BESTCHOICE มี LINE LIFF อยู่แล้วที่ `/liff/*` — ใช้ link ต่อไปนี้ใน LINE OA:

| หน้า LIFF | ใช้เมื่อ |
|-----------|---------|
| `/liff/register` | ลูกค้าใหม่ต้องการสมัคร |
| `/liff/contract` | ดูรายละเอียดสัญญา |
| `/liff/history` | ดูประวัติการชำระ |
| `/liff/early-payoff` | ชำระก่อนกำหนด |
| `/liff/profile` | ข้อมูลโปรไฟล์ลูกค้า |

### ขั้นตอน Link LIFF ใน Rich Menu

1. ไปที่ LINE Developers Console → เลือก Channel
2. เปิด LIFF tab → Copy LIFF URL (`https://liff.line.me/[LIFF_ID]`)
3. ตั้ง action ปุ่ม Rich Menu เป็น URI → วาง LIFF URL
4. Test บน mobile ว่า LIFF เปิดได้ใน LINE
