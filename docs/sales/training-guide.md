# คู่มือฝึกอบรมพนักงานขาย — BESTCHOICE

**เวอร์ชัน**: 1.0
**อัปเดต**: 2026-04-04
**เจ้าของ**: CSO — BESTCHOICE
**กลุ่มเป้าหมาย**: พนักงานขาย LINE OA + พนักงานหน้าร้าน (Walk-in)

---

## ส่วนที่ 1: รู้จัก BESTCHOICE (ก่อนเริ่มขาย)

### เราขายอะไร

| สินค้า | มือ1 (ใหม่) | มือ2 |
|--------|------------|------|
| iPhone | ✅ ขาย | ✅ ขาย (คัดสภาพดี) |
| iPad | ✅ ขาย | ❌ ไม่ขาย |
| Android / Samsung / อื่นๆ | ❌ ไม่ขาย | ❌ ไม่ขาย |

> **จำให้ขึ้นใจ**: iPhone + iPad มือ1 เท่านั้น + iPhone มือ2 — ไม่มีอื่น

### 5 จุดขายหลัก (ท่องได้เลย)

1. **บัตรประชาชนใบเดียว** — ไม่ต้องมีบัตรเครดิต ไม่ต้องมีผู้ค้ำ
2. **ไม่เช็คเครดิตบูโร** — ประวัติไม่ดีก็ผ่อนได้
3. **ทุกอาชีพผ่อนได้** — พนักงาน นักศึกษา ฟรีแลนซ์ แม่ค้า เกษตรกร
4. **ดาวน์ถูก ค่างวดต่ำที่สุด** — ผ่อนได้ถึง 24 เดือน
5. **อนุมัติใน 15 นาที** — เอกสารครบ รู้ผลทันที

## ส่วนที่ 2: Script หลัก (4 ขั้นตอน)

### ขั้นตอนที่ 1: ทักทาย (Greet)

**LINE OA:**
```
สวัสดีค่ะ/ครับ ยินดีต้อนรับสู่ BESTCHOICE ร้านผ่อน iPhone ลพบุรี

✅ ผ่อนได้ด้วยบัตรประชาชนใบเดียว
✅ ไม่เช็คเครดิตบูโร
✅ ทุกอาชีพผ่อนได้ — อนุมัติใน 15 นาที

สนใจ iPhone หรือ iPad รุ่นไหนเป็นพิเศษคะ/ครับ?
```

**Walk-in:**
```
สวัสดีครับ/ค่ะ ยินดีต้อนรับสู่ BESTCHOICE นะครับ/ค่ะ
วันนี้มาเพื่อดูอะไรเป็นพิเศษไหมครับ/ค่ะ? หรือให้ช่วยแนะนำก็ได้นะครับ/ค่ะ
```

---

### ขั้นตอนที่ 2: สอบถามความต้องการ (Discover)

คำถามสำคัญ 3 ข้อ:
1. "งบประมาณดาวน์ประมาณเท่าไหร่ครับ/ค่ะ?"
2. "มือหนึ่งหรือมือสองก็ได้ไหมครับ/ค่ะ?"
3. "อาชีพอะไรครับ/ค่ะ? (เพื่อแนะนำเอกสารที่เหมาะสม)"

**เคล็ดลับ:** ฟังก่อนพูด — เข้าใจความต้องการก่อนเสนอ

---

### ขั้นตอนที่ 3: เสนอสินค้า (Recommend)

**แนวทางเลือกรุ่นตามงบ:**

| งบดาวน์ | แนะนำ |
|---------|-------|
| < 3,000 บาท | iPhone มือ2 รุ่นเก่า |
| 3,000 – 6,000 บาท | iPhone 14 หรือ iPhone 15 |
| 6,000+ บาท | iPhone 15 Pro / iPhone 16 |

**เสนอไม่เกิน 2–3 รุ่น** — มากเกินทำให้ลูกค้าลังเล

**ตัวอย่าง Script เสนอ (iPhone 16):**
```
iPhone 16 รุ่นล่าสุด ราคาเต็ม 32,900 บาท

ตัวเลือกผ่อน:
• 12 เดือน — ดาวน์ 3,000 | งวดละ 2,600 บาท
• 18 เดือน — ดาวน์ 2,000 | งวดละ 1,800 บาท
• 24 เดือน — ดาวน์ 1,500 | งวดละ 1,400 บาท
```

---

### ขั้นตอนที่ 4: CTA — ปิดการขาย (Close)

**เมื่อลูกค้าสนใจ — ขอเอกสาร:**
```
เตรียมเอกสารได้เลยนะคะ/ครับ:
• บัตรประชาชน (ถ่ายรูปหน้า-หลัง)
• สลิปเงินเดือน หรือหลักฐานรายได้ 2-3 เดือน

ส่งมาทาง LINE นี้เลยคะ/ครับ
หรือมาที่ร้าน BESTCHOICE ลพบุรี (เปิด 9:00–19:00 ทุกวัน)
```

**สมัครออนไลน์:**
```
1. กรอกฟอร์มที่ bestchoicephone.app
2. อัปโหลดบัตรประชาชน + หลักฐานรายได้
3. เราติดต่อกลับภายใน 15 นาที
4. จ่ายดาวน์ + รับเครื่อง
```

## ส่วนที่ 3: Objection Handling (รับมือข้อโต้แย้ง)

### "ค่างวดแพงไป"
```
เข้าใจเลยคะ/ครับ — ลองปรับผ่อนนานขึ้นได้นะ
เช่น iPhone 15 ผ่อน 24 เดือน งวดละแค่ 1,200 บาทเอง
หรืออยากเปลี่ยนรุ่นที่เบากว่า บอกได้เลยคะ/ครับ
```

### "กลัวไม่ผ่าน / เครดิตไม่ดี"
```
ไม่ต้องกังวลเลยคะ/ครับ — เราไม่เช็คเครดิตบูโร
สิ่งสำคัญคือหลักฐานรายได้เท่านั้น
ลองส่งเอกสารมาให้ดูก่อนได้เลย ฟรี ไม่มีค่าใช้จ่าย
```

### "อยากได้ Android / Samsung"
```
ขอโทษนะคะ/ครับ BESTCHOICE เชี่ยวชาญเฉพาะ iPhone และ iPad
ไม่มี Android ในสต็อกคะ/ครับ

แต่ถ้ายังเปิดรับ iPhone — เรามีทุกรุ่น ทุกราคา
สนใจให้แนะนำในงบที่มีไหมคะ/ครับ?
```

### "ขอคิดดูก่อน"
```
ได้เลยคะ/ครับ ไม่ต้องรีบ
ถ้ามีคำถามเพิ่มเติม ทักมาได้ตลอดเลย
ขอบคุณที่สนใจ BESTCHOICE นะคะ/ครับ
```

### "เคยโดนปฏิเสธจากที่อื่น"
```
เสียใจด้วยนะคะ/ครับ
BESTCHOICE ไม่ได้ใช้เกณฑ์เดียวกับธนาคาร
เราดูรายได้จริง ไม่ใช่ประวัติเครดิต
ลองส่งเอกสารมาประเมินก่อนได้เลยคะ/ครับ ฟรี
```

## ส่วนที่ 4: เอกสารตามอาชีพ

| อาชีพ | เอกสารหลัก | เอกสารเสริม |
|-------|-----------|------------|
| พนักงาน / ข้าราชการ | บัตร ปชช. + สลิปเงินเดือน | ใบรับรองเงินเดือน |
| ฟรีแลนซ์ / แม่ค้า | บัตร ปชช. + Statement บัญชี | ใบเสร็จ / สัญญาจ้าง |
| นักศึกษา | บัตร ปชช. + บัตรนักศึกษา | หลักฐานจากผู้ปกครอง |
| เกษตรกร | บัตร ปชช. + Statement บัญชี | หลักฐานรายได้ฟาร์ม |

**เคล็ดลับ:** เอกสารยิ่งครบ — อนุมัติยิ่งเร็ว

## ส่วนที่ 5: สิ่งที่ห้ามพูด / ห้ามทำ (กฎเหล็ก)

### ❌ ห้ามพูดเด็ดขาด

| อย่าพูด | เพราะ | พูดแทนว่า |
|---------|-------|-----------|
| "มี Android ด้วยนะ" | เราไม่ขาย Android | "เราเชี่ยวชาญเฉพาะ iPhone และ iPad ครับ/ค่ะ" |
| "iPad มือ2 ก็มีนะ" | ไม่มีใน portfolio | "เรามีเฉพาะ iPad มือ1 ใหม่ครับ/ค่ะ" |
| "ราคาตกลงกันได้" | ลดราคาไม่ได้ผ่านหัวหน้า | "ราคาตรึงตามระบบ แต่แถม accessories ได้ครับ/ค่ะ" |
| "อนุมัติ 100% แน่นอน" | ให้สัญญาเกินจริง | "โอกาสผ่านสูงมาก ถ้าเอกสารครบครับ/ค่ะ" |
| "ไม่มีดอกเบี้ย" | ถ้าไม่ใช่ความจริง | บอกข้อมูลตามจริงในระบบ |

### ❌ ห้ามทำเด็ดขาด

- อย่า pressure หรือบีบบังคับลูกค้า
- อย่าลดราคาเครื่องโดยไม่ผ่านหัวหน้า
- อย่าส่งข้อความติดตามเกิน 3 ครั้ง (PDPA)
- อย่าแชร์ข้อมูลลูกค้าให้บุคคลภายนอก
- อย่าให้ข้อมูลนอกระบบ (ราคา / เงื่อนไขที่ไม่ตรงกับระบบ)

## ส่วนที่ 6: Upsell (เพิ่มยอดขาย)

**ลำดับการเสนอ:**
1. **เคส + ฟิล์มกันรอย** — เสนอก่อนเสมอ (อัตราสำเร็จสูงสุด)
2. **ประกันภัยมือถือ** — เสนอระหว่างรอผลอนุมัติ
3. **AirPods / Apple Watch** — เสนอหลังปิดการขายหลัก

**Script Upsell:**
```
ระหว่างรอผล ขอแนะนำ package เสริมนะครับ/ค่ะ:

Package A — ป้องกันพื้นฐาน
• เคสกันกระแทก + ฟิล์มกระจก — ติดให้ฟรีก่อนออกร้าน

Package B — คุ้มครองครบ (เพิ่มต่อปี)
• ประกันจอแตก + น้ำเข้า + เคส + ฟิล์ม

สนใจ package ไหนครับ/ค่ะ?
```

## ส่วนที่ 7: KPI พนักงานขาย

| KPI | เป้าหมาย | วิธีวัด |
|-----|----------|--------|
| Conversion Rate (Online) | ≥ 10% | สอบถาม → สัญญา |
| Conversion Rate (Walk-in) | ≥ 30% | Walk-in → สัญญา |
| Response Time (LINE OA) | ≤ 5 นาที | นับจากข้อความเข้า |
| Upsell Rate | ≥ 40% | สัญญาที่มี upsell ≥ 1 รายการ |
| Doc Completeness | 100% | บันทึกครบทุก field ในระบบ |

## ส่วนที่ 8: Escalation — เมื่อไรต้องบอกหัวหน้า

| สถานการณ์ | ทำอะไร |
|----------|--------|
| ลูกค้าขอลดราคาเครื่อง | แจ้ง OWNER / BRANCH_MANAGER |
| ปัญหาการอนุมัติผิดปกติ | แจ้งทีม Credit ทันที |
| ลูกค้าร้องเรียน / ไม่พอใจ | แจ้งหัวหน้าหน้าร้านทันที |
| ระบบ BESTCHOICE มีปัญหา | แจ้ง IT ผ่าน LINE กลุ่มทีม |
