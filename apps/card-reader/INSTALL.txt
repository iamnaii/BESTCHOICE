  วิธีติดตั้ง (ทำครั้งเดียว)
  ─────────────────────────
  1. แตกไฟล์ zip นี้ไปไว้ที่ไหนก็ได้ (เช่น Desktop หรือ Documents)
  2. เปิดโฟลเดอร์ที่แตกไฟล์
  3. ดับเบิลคลิก  setup.bat
  4. ทำตามที่โปรแกรมถาม → เสร็จ!


  วิธีใช้งาน
  ──────────
  1. เสียบเครื่องอ่านบัตร USB เข้าคอมพิวเตอร์
  2. ดับเบิลคลิก "BESTCHOICE Card Reader" บน Desktop
     (หรือดับเบิลคลิก BestchoiceCardReader.bat ในโฟลเดอร์นี้)
  3. เสียบบัตรประชาชนเข้าเครื่องอ่าน
  4. ระบบจะอ่านข้อมูลจากบัตรให้อัตโนมัติ


  ถอนการติดตั้ง
  ─────────────
  ดับเบิลคลิก  uninstall.bat  แล้วลบโฟลเดอร์นี้ทิ้ง


  แก้ปัญหา
  ────────
  ❌ "ไม่พบเครื่องอ่านบัตร"
     → ตรวจสอบว่าเสียบเครื่องอ่านบัตร USB แล้ว
     → ลองถอดแล้วเสียบใหม่

  ❌ "ไม่พบ PC/SC service"
     → เปิด Services (พิมพ์ services.msc ใน Start Menu)
     → หา "Smart Card" → คลิกขวา → Start

  ❌ "เชื่อมต่อบัตรไม่ได้"
     → ดึงบัตรออกแล้วเสียบใหม่ช้าๆ
     → ตรวจสอบว่าบัตรไม่ชำรุด

  ❌ โปรแกรมไม่เปิด
     → ลองคลิกขวา BestchoiceCardReader.bat → Run as Administrator


  ต้องการ
  ──────
  - Windows 10 ขึ้นไป
  - เครื่องอ่านบัตร Smart Card USB
  - ไม่ต้องลงโปรแกรมเพิ่มเติมอื่น
