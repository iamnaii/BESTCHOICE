# SOP: Customer Service via LINE
**เวอร์ชัน:** 1.1
**เจ้าของ:** COO / Customer Service Agent (น้องเบส)
**อัปเดตล่าสุด:** 2026-04-02
**SLA:** ตอบกลับภายใน **5 นาที** (เวลาทำการ 09:00–18:00)

> **สินค้าที่จำหน่าย:** iPhone มือ1 / มือ2 และ iPad มือ1 เท่านั้น
> **จุดขายหลัก:** ผ่อนด้วยบัตรประชาชน | ไม่เช็คเครดิตบูโร | อนุมัติภายใน 15 นาที

---

## 1. ขอบเขตและความรับผิดชอบ

| บทบาท | ความรับผิดชอบ |
|-------|--------------|
| Customer Service (น้องเบส) | รับ-ตอบ LINE ทุกข้อความ, ระบุประเภทคำถาม |
| SALES | ตอบคำถามเรื่องผลิตภัณฑ์, ราคา, โปรโมชัน |
| BRANCH_MANAGER | อนุมัติข้อยกเว้น, escalation level 2 |
| COO | escalation level 3, policy decisions |

---

## 2. ขั้นตอนการรับและตอบ LINE

### 2.1 เมื่อได้รับข้อความ (T=0)
1. รับทราบข้อความทันที — ส่ง auto-reply: *"สวัสดีครับ/ค่ะ ได้รับข้อความแล้ว จะตอบกลับภายใน 5 นาทีครับ/ค่ะ"*
2. ระบุประเภทคำถาม (ดู Section 3)
3. ตอบหรือ route ไปยังทีมที่ถูกต้อง

### 2.2 SLA Timer
- **T+3 นาที:** ถ้ายังไม่ได้ตอบ → notify SALES/agent ที่รับผิดชอบ
- **T+5 นาที:** SLA breach → escalate ให้ BRANCH_MANAGER ทราบทันที
- **T+10 นาที:** ไม่มีการตอบ → COO รับทราบและดำเนินการ

---

## 3. การแยกประเภทคำถาม

### Level 1 — Customer Service ตอบได้เลย (T ≤ 2 นาที)
- สถานะงวดค้างชำระ
- วันครบกำหนดชำระ
- ยอดคงเหลือสัญญา
- ข้อมูลสาขา/เวลาทำการ
- สถานะคำขอสินเชื่อ (pending/approved/rejected)

**Template:**
```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]
[คำตอบ]
หากมีข้อสงสัยเพิ่มเติม ติดต่อได้เลยนะครับ/ค่ะ
BESTCHOICE Team 🙏
```

### Level 2 — Route ให้ SALES (T ≤ 5 นาที)
- สอบถามผลิตภัณฑ์ใหม่ (iPhone / iPad มือ1 เท่านั้น)
- ต่อรองราคา/ดาวน์
- โปรโมชัน/แคมเปญ
- ขอเพิ่มวงเงิน
- ถามเรื่อง Android หรือสินค้านอก scope → ตอบ redirect ก่อน (ดู FAQ Section 4.4)

**Handoff message:**
```
ขอบคุณครับ/ค่ะ กำลังส่งเรื่องให้ทีมขายดูแลต่อนะครับ/ค่ะ
จะได้รับการติดต่อกลับภายใน 5 นาที
```

### Level 3 — Escalate ให้ BRANCH_MANAGER
- ร้องเรียนเรื่องการบริการ
- ขอยกเว้นดอกเบี้ย/ค่าปรับ
- ขอแก้ไขสัญญา
- กรณีพิพาทการชำระเงิน

### Level 4 — Escalate ให้ COO (ภายใน 1 ชม.)
- ร้องเรียนซ้ำที่ยังไม่ได้รับการแก้ไข
- ภัยคุกคามทางกฎหมาย
- สื่อมวลชนหรือหน่วยงานภาครัฐ

## 4. FAQ Templates

### คำถาม: ยอดค้างชำระ
```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]
ยอดค้างชำระของคุณมีดังนี้:
- งวดที่ [X]: [จำนวน] บาท (ครบกำหนด [วันที่])
- งวดที่ [X]: [จำนวน] บาท (ครบกำหนด [วันที่])
รวมทั้งสิ้น [รวม] บาท
ชำระได้ที่ [ช่องทาง] ครับ/ค่ะ
```

### คำถาม: สถานะอนุมัติ
```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]
ขณะนี้คำขอของคุณอยู่ในสถานะ: [สถานะ]
[ถ้า pending]: คาดว่าจะทราบผลภายใน [X] นาที
[ถ้า approved]: ยินดีด้วยครับ/ค่ะ! กรุณาติดต่อสาขาเพื่อดำเนินการต่อ
[ถ้า rejected]: เราขอโทษที่ไม่สามารถอนุมัติได้ในครั้งนี้ สามารถสอบถามเพิ่มเติมได้ครับ/ค่ะ
```

### 4.3 คำถาม: สินค้าที่จำหน่าย / ผ่อนได้อะไรบ้าง
```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]
BESTCHOICE จำหน่ายและรับผ่อนชำระสินค้าดังนี้ครับ/ค่ะ:

📱 iPhone มือ 1 (ใหม่จาก Apple)
📱 iPhone มือ 2 (คัดเกรดคุณภาพ)
📲 iPad มือ 1 (ใหม่จาก Apple)

✅ ผ่อนด้วยบัตรประชาชนเพียงใบเดียว
✅ ไม่เช็คเครดิตบูโร
✅ อนุมัติภายใน 15 นาที

สนใจสินค้ารุ่นไหนเป็นพิเศษไหมครับ/ค่ะ?
```

### 4.4 คำถาม: Android / สินค้านอก scope
```
สวัสดีครับ/ค่ะ คุณ [ชื่อ]
ขอบคุณที่สนใจสินค้าของเรานะครับ/ค่ะ 🙏

ขณะนี้ BESTCHOICE เชี่ยวชาญด้าน iPhone และ iPad โดยเฉพาะครับ/ค่ะ
เราจึงยังไม่มีบริการสำหรับ [Android/สินค้าที่ถาม] ในขณะนี้

หากสนใจ iPhone หรือ iPad สามารถสอบถามเพิ่มเติมได้เลยนะครับ/ค่ะ
เรายินดีให้คำแนะนำเรื่องรุ่นที่เหมาะกับการใช้งานของคุณครับ/ค่ะ 😊
```
