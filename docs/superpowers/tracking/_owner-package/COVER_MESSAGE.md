## 🎯 หลังจาก AI ตอบกลับ → Owner ต้องทำอะไรเรื่อ

### ตอบ Pre-flight Check 5 ข้อ
AI จะส่ง 5 คำถามมา ตัวอย่าง:
1. ✅ ฉมอ่าน README ครบแล้ว
2. ✅ ฉมเจอ conflict ที่ X — confirm ใช้ใหม?
3. ✅ Codebase ที่ branch ไหน?
4. ✅ Deadline จริงคือเมื่อไหร่?
5. ✅ ทำ Action Items ก่อน หรือ Settings Audit ก่อน?

**Owner ตอบทั้ง 5 ข้อในข้อความเดียว** → แล้ว AI จะเริ่ม Phase 1
