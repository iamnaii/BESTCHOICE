# Post 2 — TikTok เสียงลูกค้า (Testimonial Pillar)

**Platform:** TikTok
**วันที่โพสต์:** เสาร์ 5 เมษายน 2026
**Format:** POV / Lifestyle
**ความยาว:** 45-60 วินาที
**Pillar:** เสียงลูกค้า
**CTA:** LINE OA

---

## Script / Storyboard

### Hook (0-3 วินาที) — ต้องดึงดูดทันที
**ภาพ:** มือถือใหม่ในกล่อง กำลังเปิดกล่อง
**Text overlay:** `"รับ iPhone ใหม่ ใช้แค่บัตรปชช ใบเดียว 👀"`
**เสียง:** เสียงดังตื่นเต้น หรือเพลง trending ที่ฮิตช่วงนั้น

---

### Scene 1 (3-15 วินาที) — สถานการณ์ลูกค้า
**ภาพ:** ลูกค้า (หรือ staff แสดงเป็นลูกค้า) ถือบัตรประชาชนใบเดียว
**Caption/Text overlay:**
```
เมื่อเช้า: ยื่นบัตรปชช
```
**Voiceover/ข้อความ:** "ปกติคิดว่าต้องมีบัตรเครดิต หรือคนค้ำ — แต่ที่ BESTCHOICE ใช้แค่บัตรปชช ใบเดียว"

---

### Scene 2 (15-30 วินาที) — กระบวนการง่าย
**ภาพ:** แสดง 3 ขั้นตอนแบบ fast-cut
1. ยื่นบัตรปชช
2. รอ 15 นาที
3. รับเครื่อง!

**Text overlay:**
```
ขั้นตอนที่ 1: ยื่นบัตรปชช ✅
ขั้นตอนที่ 2: รอผล 15 นาที ✅
ขั้นตอนที่ 3: รับเครื่องเลย! ✅
```

---

### Scene 3 (30-50 วินาที) — ผลลัพธ์ / ความสุข
**ภาพ:** ลูกค้าถือ iPhone ใหม่ยิ้มแย้ม ถ่ายรูปกับเครื่อง
**Text overlay:**
```
บ่ายวันเดียวกัน: iPhone ใหม่อยู่ในมือแล้ว 🎉
ผ่อนเดือนละ XXX บาท*
```
**Voiceover:** "ไม่เชื่อก็ต้องเชื่อ รู้ผลวันเดียว รับเครื่องเลย"

---

### CTA (50-60 วินาที)
**ภาพ:** หน้า LINE OA หรือ QR code
**Text overlay:**
```
อยากได้ iPhone ใหม่?
ทักมาที่ LINE OA: @bestchoice
หรือ bestchoicephone.app
```
**Voiceover:** "ทักมาก่อนได้เลย ฟรี! ไม่มีข้อผูกมัด"

---

## Caption TikTok

```
POV: ตื่นมาอยากได้ iPhone ใหม่ บ่ายนี้รับเครื่องเลย 📱✨

ใช้แค่บัตรปชช ใบเดียว ไม่เช็คเครดิตบูโร รู้ผลใน 15 นาที

ทักมาที่ LINE OA: @bestchoice
🌐 bestchoicephone.app

#BestChoice #ผ่อนiPhone #ลพบุรี #iPhone #POV #ผ่อนมือถือ #ไม่เช็คเครดิตบูโร
```

---

## Visual Notes

- ถ่ายในร้าน BESTCHOICE จริง — ให้ดู authentic
- แสงสว่าง ชัดเจน ไม่มืด
- ใช้เพลง trending ที่ไม่มีลิขสิทธิ์ปัญหา
- Sub-text ราคาต้องได้รับการยืนยันจากฝ่ายขายก่อน

---

## Checklist ก่อนโพสต์

- [ ] ลูกค้าให้ consent การปรากฏใน content (ถ้าใช้ลูกค้าจริง)
- [ ] ฝ่ายขายยืนยันราคาผ่อนในคลิป
- [ ] เพลง background ไม่มีปัญหา copyright
- [ ] ความยาวไม่เกิน 60 วินาที
