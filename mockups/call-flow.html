1. UI ในแชท: ปุ่มโทรออก 2 จุด

A. เบอร์โทรใน panel ลูกค้า (idle)

ส

สมชาย ใจดี

เสี่ยงสูง

→ Hover ที่เบอร์ → text เปลี่ยนเป็นสีเขียว, คลิกเพื่อโทร

B. ปุ่ม "ดำเนินการ" → popover

→ "โทรออก" ขึ้นบนสุด เห็นชัด มีเส้นแยกจากกลุ่มล่าง

2. หลังกด → State เปลี่ยนทันที

Idle

พร้อมรับการคลิก

Calling — 1-3 วินาที

รอ Yeastar ตอบรับคำสั่ง

กำลังโทรออก — รับสายจากโทรศัพท์ของคุณ

Connected — toast ขึ้น

ปลดล็อกอัตโนมัติใน 10 วินาที

3. กลไกจริง — เกิดอะไรเบื้องหลัง

click-to-dial ไม่ได้โทรจาก browser/web; Yeastar PBX โทรหาเบอร์ภายในของพนักงานก่อน แล้วค่อย bridge ไปลูกค้า

1. พนง. คลิก

"โทรออก" ในแชท

→ POST /yeastar/call/originate

2. Yeastar PBX

โทรเข้าเบอร์ภายใน

ของพนง. (ext 101)

3. โทรศัพท์ดัง

IP phone / softphone

ของพนง. → รับสาย

4. Bridge อัตโนมัติ

PBX โทรเข้าเบอร์ลูกค้า

081-234-5001

5. คุยกับลูกค้า

บันทึกเสียงอัตโนมัติ

→ CallLog auto-create

หลังวางสาย:

• Yeastar ส่ง CDR (call detail record) เข้า webhook /yeastar/webhook/cdr

• ระบบสร้าง CallLog row อัตโนมัติ — direction, duration, recording URL

• ขึ้นใน "ประวัติโทร" ของลูกค้า + "การชำระล่าสุด" บน Customer 360 panel

• ถ้ารับสายไม่ทัน → noAnswerCount เพิ่ม → escalation guardrail trigger

4. กรณีโทรไม่ออก

ไม่ได้ตั้งค่าเบอร์ภายใน — กรุณาผูก extension ที่ "ตั้งค่า → โปรไฟล์"

พนง. ยังไม่ได้ผูก extension

ไม่มีสัญญาที่ใช้งาน — โทรออกผ่าน Yeastar ต้องระบุสัญญา

ลูกค้าไม่มี active contract

โทรออกไม่สำเร็จ — Yeastar ตอบ 503
