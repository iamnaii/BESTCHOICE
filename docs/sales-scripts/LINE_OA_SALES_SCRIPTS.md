# LINE OA Sales Scripts & FAQ — BESTCHOICE Online Sales

## ช่องทาง
- **LINE OA**: เป็นช่องทางหลัก (1:1 messaging)
- **Facebook Messenger**: ใช้ script เดียวกัน
- **TikTok/Instagram DM**: DM ส่วนตัว → นำเข้า LINE OA

> **หมายเหตุสำคัญ**: BESTCHOICE ขายเฉพาะ **iPhone มือ1/มือ2** และ **iPad มือ1** เท่านั้น
> ไม่รับออเดอร์ Android, ไม่ขาย iPad มือ2, ไม่ขายอุปกรณ์อื่น

## Objection Handling Scripts

### กรณี: "ผ่อนแพงไปหน่อย / งวดแพงเกินไป"

```
เข้าใจเลยค่ะ/ครับ!

ลองดูตัวเลือกผ่อนนานขึ้นไหมคะ/ครับ?
ผ่อน 24 เดือน จะทำให้ค่างวดเบาลงมากเลย
เช่น iPhone 15 ผ่อน 24 เดือน งวดละแค่ประมาณ 1,200 บาท

หรือถ้าอยากดาวน์น้อยกว่านี้ บอกได้เลยคะ/ครับ
เราจะหาตัวเลือกที่เหมาะที่สุดให้
```

### กรณี: "กลัวไม่ผ่านการอนุมัติ"

```
ไม่ต้องกังวลนะคะ/ครับ!

เราไม่เช็คเครดิตบูโร และทุกอาชีพผ่อนได้

ที่สำคัญคือหลักฐานรายได้ ไม่ว่าจะเป็น:
• สลิปเงินเดือน
• Statement บัญชีธนาคาร
• หลักฐานรายได้ฟรีแลนซ์ / กิจการ

ถ้าไม่แน่ใจ ส่งเอกสารมาให้เราดูก่อนได้เลย
เราจะประเมินเบื้องต้นให้ฟรี ไม่มีค่าใช้จ่ายค่ะ/ครับ
```

### กรณี: "อยากได้ Android / Samsung / Xiaomi"

```
ขอบคุณที่สนใจนะคะ/ครับ!

ต้องขอเรียนให้ทราบว่า BESTCHOICE เชี่ยวชาญเฉพาะ iPhone และ iPad ค่ะ/ครับ
เราไม่มี Android ในสต็อก

แต่ถ้าคุณยังเปิดรับ iPhone อยู่ เรามีทุกรุ่นทุกราคา ตั้งแต่
iPhone มือสองราคาประหยัด ไปถึง iPhone 16 Pro Max รุ่นล่าสุด

สนใจให้แนะนำ iPhone ในงบที่คุณมีไหมคะ/ครับ?
```

### กรณี: "ยังไม่พร้อม / ขอคิดดูก่อน"

```
ได้เลยค่ะ/ครับ ไม่ต้องรีบตัดสินใจ

ถ้ามีคำถามเพิ่มเติม หรืออยากให้ช่วยเปรียบเทียบรุ่น
ทักมาได้ตลอดเลยนะคะ/ครับ เราพร้อมช่วยเสมอ

ขอบคุณที่สนใจ BESTCHOICE ค่ะ/ครับ
```

### กรณี: "เคยโดนปฏิเสธจากที่อื่น"

```
เสียใจด้วยนะคะ/ครับที่เจอปัญหาแบบนั้น

BESTCHOICE ไม่ได้ใช้เกณฑ์เดียวกับธนาคาร
เราพิจารณาจากรายได้จริงและความสามารถในการผ่อน
ไม่ได้ดูประวัติเครดิตบูโร

ลองส่งเอกสารมาให้เราดูก่อนได้เลยค่ะ/ครับ
เราจะประเมินให้เร็วๆ โดยไม่มีค่าใช้จ่าย
```

## FAQ — คำถามที่พบบ่อย

### ด้านราคา & เงื่อนไข

**Q: ต้องมีผู้ค้ำไหม?**
```
ไม่ต้องเลยนะคะ/ครับ
ผ่อนได้ด้วยบัตรประชาชนใบเดียว ไม่ต้องมีผู้ค้ำ ไม่ต้องมีบัตรเครดิต
```

**Q: รู้ผลอนุมัติเมื่อไร?**
```
ทราบผลภายใน 15 นาที ถ้าเอกสารครบค่ะ/ครับ
```

**Q: ดาวน์เท่าไร?**
```
ขึ้นอยู่กับรุ่นและจำนวนงวดที่เลือกค่ะ/ครับ
ยิ่งผ่อนนานยิ่งดาวน์น้อย ค่างวดก็เบาลงด้วย

อยากให้แนะนำตามงบประมาณที่มีไหมคะ/ครับ?
```

**Q: ต้องใช้บัตรเครดิตไหม?**
```
ไม่ต้องเลยค่ะ/ครับ
ผ่อนผ่าน Promptpay / โอนเงิน / หักบัญชีธนาคาร
ไม่ต้อง swipe บัตร ไม่มีดอกเบี้ยบัตรเครดิต
```

**Q: ผ่อนกี่เดือนได้?**
```
มีให้เลือก 6, 10, 12, 18, และ 24 เดือนค่ะ/ครับ
ส่วนใหญ่เลือก 10-12 เดือน เพราะดาวน์ไม่สูงและงวดสบาย
```

**Q: ผ่อนหมดก่อนกำหนดได้ไหม?**
```
ได้เลยค่ะ/ครับ ไม่มีค่าปรับ
```

### ด้านอาชีพ & สิทธิ์

**Q: ทุกอาชีพผ่อนได้จริงไหม?**
```
จริงค่ะ/ครับ ทุกอาชีพผ่อนได้

• ฟรีแลนซ์

• เกษตรกร

สิ่งสำคัญคือหลักฐานรายได้ค่ะ/ครับ
บอกอาชีพมา เราจะแนะนำเอกสารที่เหมาะสมให้
```

**Q: นักศึกษาผ่อนได้ไหม?**
```
ผ่อนได้นะคะ/ครับ

• บัตรประชาชน
• บัตรนักศึกษา
• หลักฐานรายได้หรือหลักฐานการสนับสนุนจากผู้ปกครอง
```

**Q: ฟรีแลนซ์ / แม่ค้าผ่อนได้ไหม?**
```
ผ่อนได้ค่ะ/ครับ
เอกสารรายได้ที่ใช้ได้:
• Statement บัญชีธนาคาร
• ใบเสร็จจากลูกค้า / กิจการ
• สัญญาจ้าง
• หลักฐานยอดโอนที่สม่ำเสมอ
```

### ด้านเอกสาร & กระบวนการ

**Q: ต้องใช้เอกสารอะไรบ้าง?**
```
เอกสารหลัก:
• บัตรประชาชน (ถ่ายรูปหน้า-หลัง)
• สลิปเงินเดือนหรือหลักฐานรายได้ 2-3 เดือน

เอกสารเสริม (ช่วยให้อนุมัติเร็วขึ้น):
• Statement บัญชีธนาคาร
• ใบรับรองเงินเดือน

เคล็ดลับ: เอกสารยิ่งครบ อนุมัติยิ่งเร็วค่ะ/ครับ
```

**Q: ข้อมูลส่วนตัวปลอดภัยไหม?**
```
ปลอดภัย 100% ค่ะ/ครับ
• เก็บข้อมูลเฉพาะในระบบของเรา
• เข้ารหัสเอกสารทุกชิ้น
• ปฏิบัติตาม พ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล (PDPA)
• ไม่แชร์ข้อมูลให้บุคคลภายนอก
```

**Q: ขั้นตอนการผ่อนเป็นอย่างไร?**
```
4 ขั้นตอนง่ายๆ ค่ะ/ครับ:

1. กรอกฟอร์ม + ส่งเอกสาร (ประมาณ 5 นาที)
2. เราตรวจสอบและอนุมัติ (ประมาณ 15 นาที)
3. เซ็นสัญญา (ที่ร้านหรือออนไลน์)
4. จ่ายดาวน์ + รับเครื่อง

รวมประมาณ 1 ชั่วโมง ได้เครื่องกลับบ้านเลยค่ะ/ครับ
```

| **Response Rate** | ≥ 80% | ตอบทุกข้อความที่ไม่ใช่ spam |

## Follow-up Sequences

### วันที่ 1 (หลังติดต่อครั้งแรก ยังไม่สมัคร)
```
สวัสดีค่ะ/ครับ ยังสนใจ iPhone รุ่น [รุ่น] อยู่ไหม?
มีคำถามเพิ่มเติมยินดีช่วยเลยนะคะ/ครับ
```

### วันที่ 3 (ยังไม่มีการตอบ)
```
สวัสดีค่ะ/ครับ แจ้งให้ทราบว่า [รุ่นที่สนใจ] ยังมีในสต็อกนะคะ/ครับ
ถ้าสนใจผ่อน ทักมาได้เลย เราพร้อมดูแลค่ะ/ครับ
```

### วันที่ 7 (ติดตามรอบสุดท้าย)
```
สวัสดีค่ะ/ครับ ขอถามอีกครั้งว่ายังสนใจ iPhone อยู่ไหม?
ถ้ามีปัญหาหรือข้อสงสัยใดๆ บอกได้เลยนะคะ/ครับ
ไม่ตอบก็ไม่เป็นไร ขอบคุณที่ให้โอกาสค่ะ/ครับ
```
