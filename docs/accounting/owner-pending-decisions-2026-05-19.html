C · CPA questions (4 ข้อ)

ส่งให้ CPA ตอบ — Dev รอผลก่อน implement Phase A.5-SP5/SP6

C1 · CR-001 VAT-on-interest #1

ภ.พ.30 ของบริษัทที่ผ่านมาเคย report VAT บน "ดอกเบี้ยเช่าซื้อ" ไหม?

ระบบปัจจุบัน book VAT 7% บนยอดรวม (เงินต้น + ดอกเบี้ย + ค่าคอม) แล้วยังนำส่งทาง ภ.พ.30 ทุกเดือน. คำถามนี้สำหรับยืนยันว่าวันนี้ทำอยู่ตามนี้จริง — ก่อนตัดสินใจว่าจะแก้.

C2 · CR-001 VAT-on-interest #2

ม.79/4(7) ป.รัษฎากร ใช้กับ "ดอกเบี้ยเช่าซื้อ" ในนิยามของเราหรือไม่?

มาตรา 79/4(7) ยกเว้น VAT สำหรับ "ดอกเบี้ย" — แต่ definition ของ "ดอกเบี้ย" ใน hire-purchase context เป็นอะไรกันแน่? ถือว่า interest component ของยอดผ่อนเป็น "ดอกเบี้ย" ตาม ม.79/4(7) ได้ไหม?

C3 · CR-001 VAT-on-interest #3

ถ้ายกเว้นจริง — split base ออกเป็น taxable principal + exempt interest บนใบกำกับภาษีอย่างไร?

ตัวอย่าง ค่างวด 1,515.83฿ = principal 1,416.66 + interest 0 (ปัจจุบัน) + VAT 99.17. ถ้าแยก base: ค่างวด = principal 1,416.66 + interest X (exempt) + VAT = principal × 7% เท่านั้น. CPA แนะนำ format ใบกำกับให้แสดงยอดอย่างไรเพื่อให้ผ่านการตรวจ?

C4 · N-005 Interest recognition

เคสของเรา (37.5% flat / ≈154% EIR / 12 งวด) ผ่าน NPAEs §11 simplification ได้ไหม?

ปัจจุบันใช้ straight-line allocation: interestTotal / totalMonths. TFRS 15 §60-65 ให้ใช้ Effective Interest Method (EIR) แต่ NPAEs §11 ผ่อนปรนถ้า "significant financing component" ไม่ material. คำถามนี้สำคัญ:

  • NPAEs §11 simplification ใช้ได้ในเคสเรา?
  • ถ้าใช่ — ออก Policy Memo ลายลักษณ์อักษรให้ดี เก็บไว้ audit trail
  • ถ้าไม่ใช่ — กำหนด timeline migration เป็น EIR (Dev บอก ~1 อาทิตย์)
เอกสารอ้างอิง: docs/accounting/eir-decision-memo.md (มีตารางเทียบ per-period P&L difference ระหว่าง 2 methods)
