#### Warning — W1: Unvalidated `settlementAmount` in legacy DTO path

**File**: `apps/api/src/modules/overdue/overdue.service.ts` (lines 964–965, 971–972)

```ts
// Legacy path in logContact() — uses service-level inline type, bypasses LogContactDto
settlementAmount: Number(dto.settlementAmount ?? 0),
settlementAmount: Number(dto.secondSettlementAmount ?? 0),
```

`LogContactDto` (the controller's input class) does NOT declare `settlementAmount` or `secondSettlementAmount` fields — they exist only in the service method's inline type annotation. This means:
1. These fields are never validated by class-validator (no `@IsNumber`, no `@Min`).
2. A client sending a non-numeric string causes `Number("…") = NaN`, which then reaches `new Prisma.Decimal(NaN)` and throws a 500 rather than a 400.
3. Should use `new Prisma.Decimal(String(dto.settlementAmount ?? 0))` after adding proper DTO validation.

**Fix**: Add to `LogContactDto`:
```ts
@IsOptional()
@Type(() => Number)
@IsNumber({ maxDecimalPlaces: 2 }, { message: 'settlementAmount ต้องเป็นตัวเลข' })
@Min(0.01)
settlementAmount?: number;

@IsOptional()
@Type(() => Number)
@IsNumber({ maxDecimalPlaces: 2 }, { message: 'secondSettlementAmount ต้องเป็นตัวเลข' })
@Min(0.01)
secondSettlementAmount?: number;
```

Then replace `Number(dto.settlementAmount ?? 0)` → `dto.settlementAmount` (already validated).

#### Warning — W2: `as never` cast hides type mismatch in `promise.service.ts`

**File**: `apps/api/src/modules/overdue/promise.service.ts` (line 172)

```ts
await tx.promiseSlot.createMany({
  data: slotsInput.map((s, i) => ({
    callLogId: newPromise.id,
    slotIndex: i,
    settlementDate: s.settlementDate,
    settlementAmount: s.settlementAmount as never,  // ⚠️ type cast hiding mismatch
  })),
});
```

`s.settlementAmount` is typed as `number | string` but `PromiseSlot.settlementAmount` expects `Decimal`. The `as never` suppresses TypeScript's error. Prisma handles the conversion at runtime but the cast is a maintenance smell.

**Fix**: Change `settlementAmount: s.settlementAmount as never` → `settlementAmount: new Prisma.Decimal(String(s.settlementAmount))`.
