นโยบายความเป็นส่วนตัว

BESTCHOICE

ปรับปรุงล่าสุด: 9 พฤษภาคม 2569

1. บทนำ

บริษัท เบสช้อยส์ จำกัด ("บริษัท", "เรา") ให้ความสำคัญกับการคุ้มครองข้อมูลส่วนบุคคลของลูกค้า ตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) นโยบายนี้อธิบายวิธีการเก็บรวบรวม ใช้ และเปิดเผยข้อมูลส่วนบุคคลของท่าน รวมถึงข้อมูลที่เราได้รับผ่านแพลตฟอร์ม Facebook

2. ข้อมูลที่เราเก็บรวบรวม

  • ชื่อ-นามสกุล, เลขบัตรประชาชน, ที่อยู่, เบอร์โทรศัพท์
  • ข้อมูลสัญญาเช่าซื้อและประวัติการชำระเงิน
  • ข้อมูลการติดต่อผ่าน LINE, Facebook Messenger และช่องทางอื่น
  • ข้อมูลอุปกรณ์ (IMEI, รุ่น, สี) สำหรับสินค้าที่อยู่ระหว่างผ่อนชำระ
  • รูปถ่ายสลิปการโอนเงิน

3. วัตถุประสงค์ในการใช้ข้อมูล

  • จัดทำสัญญาเช่าซื้อและบริหารจัดการการผ่อนชำระ
  • ยืนยันตัวตนและตรวจสอบเครดิต
  • แจ้งเตือนค่างวด, ใบเสร็จ, และข้อมูลสัญญา
  • ให้บริการลูกค้าผ่านระบบแชทอัตโนมัติและพนักงาน
  • ปรับปรุงบริการและวิเคราะห์ข้อมูลเพื่อพัฒนาธุรกิจ
  • ปฏิบัติตามกฎหมายและข้อบังคับที่เกี่ยวข้อง

4. การเปิดเผยข้อมูล

เราอาจเปิดเผยข้อมูลส่วนบุคคลของท่านแก่บุคคลภายนอกในกรณีดังนี้:

  • ผู้ให้บริการชำระเงิน (PaySolutions) เพื่อประมวลผลการชำระ
  • ผู้ให้บริการ LINE และ Facebook เพื่อส่งข้อความแจ้งเตือนและให้บริการลูกค้า
  • หน่วยงานราชการตามที่กฎหมายกำหนด

เราไม่ขายข้อมูลส่วนบุคคลของท่านให้บุคคลภายนอกใด ๆ และไม่ใช้ข้อมูลของท่านเพื่อการตลาดของบุคคลภายนอก

5. ระยะเวลาการเก็บรักษา

เราจัดเก็บข้อมูลส่วนบุคคลตลอดระยะเวลาที่ท่านเป็นลูกค้า และอีก 10 ปีหลังสัญญาสิ้นสุด ตามข้อกำหนดทางบัญชีและภาษี ข้อมูลแชทจัดเก็บ 6 เดือน, บันทึกการตรวจสอบ 1-2 ปี

6. สิทธิของเจ้าของข้อมูล

ท่านมีสิทธิตาม PDPA ดังนี้:

  • สิทธิในการเข้าถึงข้อมูลส่วนบุคคลของท่าน
  • สิทธิในการแก้ไขข้อมูลให้ถูกต้อง
  • สิทธิในการขอลบข้อมูล (ภายใต้เงื่อนไขที่กฎหมายกำหนด)
  • สิทธิในการคัดค้านการประมวลผล
  • สิทธิในการถอนความยินยอม
  • สิทธิในการขอให้โอนย้ายข้อมูล

7. การรักษาความปลอดภัย

เราใช้มาตรการรักษาความปลอดภัยที่เหมาะสม รวมถึงการเข้ารหัสข้อมูล (AES-256) การจำกัดสิทธิ์การเข้าถึง และการตรวจสอบบันทึกการใช้งาน เพื่อป้องกันการเข้าถึง เปลี่ยนแปลง หรือเปิดเผยข้อมูลโดยไม่ได้รับอนุญาต ข้อมูลของท่านถูกจัดเก็บบน Google Cloud Platform (asia-southeast1, สิงคโปร์)

8. ข้อมูลที่ได้รับผ่านแพลตฟอร์ม Facebook

BESTCHOICE ใช้ Facebook Pages, Messenger และ Marketing API เพื่อให้บริการลูกค้าและวิเคราะห์ประสิทธิภาพโฆษณา เมื่อท่านมีปฏิสัมพันธ์กับเพจ Facebook ของเราหรือโฆษณาที่บูสต์ เราอาจเก็บรวบรวมข้อมูลต่อไปนี้ผ่าน Facebook Graph API

8.1 ข้อมูลที่เราเก็บรวบรวม

ข้อมูล เก็บเมื่อใด วัตถุประสงค์ ระยะเวลาเก็บ
Facebook Page-Scoped User ID (PSID) เมื่อท่านส่งข้อความหาเพจของเรา ตอบกลับภายในกรอบเวลา 24 ชั่วโมง ตลอดการสนทนา + 90 วัน
ชื่อโปรไฟล์และรูปโปรไฟล์ (PSID-scoped) เมื่อท่านส่งข้อความหาเพจของเรา แสดงในกล่องข้อความสำหรับพนักงานอ้างอิง ตลอดการสนทนา + 90 วัน
เนื้อหาข้อความ เมื่อท่านส่งข้อความหาเพจของเรา เก็บประวัติบทสนทนาเพื่อให้ AI ตอบได้ตรงบริบท ตลอดการสนทนา + 90 วัน
คอมเมนต์และรีแอ็กชันบนโพสต์/โฆษณาของเรา เมื่อท่านคอมเมนต์หรือกด reaction ให้บริการลูกค้า, วิเคราะห์ sentiment, AI ตอบอัตโนมัติ คอมเมนต์ 90 วัน, ยอดรวม reaction 365 วัน
ตัวเลขประสิทธิภาพโฆษณา (รวมยอด) ต่อเนื่องตลอดที่โฆษณายังรัน ปรับปรุงการ targeting (ไม่มีข้อมูลรายบุคคล) 365 วัน (ข้อมูลรวมที่ลบตัวตนแล้ว)

8.2 เราใช้ข้อมูลอย่างไร

  • การตอบ Messenger: ผู้ช่วย AI ของเรา ("BESTCHOICE Sales Bot") จะตอบคำถามลูกค้าเกี่ยวกับแผนผ่อนชำระ, การชำระเงิน, และสินค้าที่มี ภายในกรอบเวลา 24 ชั่วโมง (Standard Messaging Window) ของ Facebook เราไม่ส่งข้อความโปรโมชันโดยที่ลูกค้าไม่ได้ขอ และไม่ใช้ message tag ที่ถูก deprecate (ACCOUNT_UPDATE, CONFIRMED_EVENT_UPDATE, POST_PURCHASE_UPDATE)
  • การจัดการคอมเมนต์: เราอ่านคอมเมนต์บนโพสต์และโฆษณาของเรา เพื่อตอบคำถามลูกค้า, จัดหมวด intent (สอบถามราคา, ตำหนิ, สแปม) และตอบกลับสาธารณะด้วยข้อมูลสินค้าที่เกี่ยวข้อง เนื้อหาที่ไม่เหมาะสมจะถูกซ่อน และจะไม่ถูกลบหากพนักงานยังไม่ได้ตรวจสอบ
  • Ad Insights: เราอ่านตัวเลขประสิทธิภาพโฆษณาแบบรวม (impressions, clicks, CTR, breakdown ตามกลุ่มประชากร) เพื่อเรียนรู้ว่าแคมเปญใดได้ผลตอบรับดีจากลูกค้า ข้อมูล insights เป็นข้อมูลรวมเท่านั้น ไม่มีข้อมูลรายบุคคล
  • Business Manager: เราอ่านรายการ ad account และเพจที่ Business Manager ของเราเป็นเจ้าของ เพื่อแสดงเป็นตัวเลือกใน admin panel ของเรา เราไม่แก้ไขทรัพย์สินใน Business Manager ผ่านแอปนี้

8.3 สิ่งที่เราไม่ทำ

  • ไม่แชร์ข้อมูล Facebook ของท่านให้บุคคลภายนอก
  • ไม่ขายข้อมูล Facebook ของท่าน
  • ไม่ใช้ข้อมูล Facebook ของท่านเพื่อยิงโฆษณาให้คนอื่น
  • ไม่โพสต์ในนามท่าน, ไม่กดถูกใจในนามท่าน, ไม่ใช้รายชื่อเพื่อนของท่าน
  • ไม่คอมเมนต์บนโพสต์ของเพจ/ผู้ใช้อื่น — เฉพาะโพสต์ของเพจที่เราดูแลเท่านั้น
  • ไม่สร้าง, แก้ไข หรือลบโฆษณาผ่าน API (จัดการผ่าน Ads Manager UI ด้วยมือเท่านั้น)

  • อีเมล: ส่งคำขอลบข้อมูลมาที่ akenarin.ak@gmail.com

  • ติดตามสถานะการลบ: https://bestchoicephone.app/privacy.html?deletion={confirmation_code}
