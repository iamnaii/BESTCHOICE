5Checklist สำหรับนักบัญชีตรวจสอบ

  1. ทุกรายการ Dr = Cr ระบบ throw error ทันทีถ้า Dr ≠ Cr (ไม่อนุญาตให้ post รายการที่ไม่ balance)
  2. ลูกหนี้ Gross (11-2101) ลดลงตรงตามค่างวด ROUND_DOWN ทุกงวดต้องลดลง 1,416.66 บาท · งวดสุดท้ายต้องเป็น 0
  3. รายได้ดอกเบี้ย (41-1101) สะสมตรงกับยอดรวมในสัญญา ตอนจบสัญญาต้องเท่ากับ 6,000 บาท พอดี (ห้ามเกินห้ามขาด)
  4. VAT ขาย ภ.พ.30 (21-2101) สะสมตรงกับ VAT ในสัญญา ตอนจบสัญญาต้องเท่ากับ 1,190 บาท พอดี · ตรงกับยอดที่นำส่งสรรพากร
  5. บัญชี Park (11-2106 ดอกเบี้ยรอตัด, 21-2102 VAT รอเรียก) ต้องเป็น 0 ตอนจบสัญญา ถ้ายังคงเหลือ = มีรายการที่ยังไม่ recognize หรือ accrual หาย
  6. ค่าคอมมิชชั่นและยอดจัดล้างหมดทันทีหลังจ่ายหน้าร้าน 21-1101 และ 21-1102 ต้องเป็น 0 หลังการโอนเงินให้ SHOP
  7. ส่วนลด/ปัดเศษ ≤ 1 บาท เท่านั้น ใหญ่กว่า 1 บาท ระบบ reject ทันที · Underpay ต้องมีผู้อนุมัติ (audit log บันทึก)
  8. VAT 60 วันบังคับ ทำงานทุกวันเวลา 02:00 น. เมื่อลูกค้าจ่ายภายหลัง ระบบกลับรายการอัตโนมัติ — ตรวจสอบในรายงาน VAT ภ.พ.30 ปลายเดือน
  9. ทุก Payment ต้องระบุบัญชีเงินสดหรือธนาคารที่รับเงิน เลือกได้ 6 บัญชี: เงินสด 3 (สุทธินีย์/เอกนรินทร์/พนักงานบัญชี) + ธนาคาร 3 (KBank/SCB ค่าใช้จ่าย/SCB ค่าเสื่อม)
  10. งบทดลอง (Trial Balance) ออกได้ทุกวัน ระบบรวมยอดทุก 2 หลักของรหัสบัญชี · Dr รวม = Cr รวม เสมอ (ถ้าไม่เท่า = มีรายการเสียหาย)
