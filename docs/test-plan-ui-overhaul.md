# แผนเทส UI Redesign — Minimal Zinc + Emerald Accent

> Design spec: [`docs/superpowers/specs/2026-04-16-shadcn-ui-redesign.md`](superpowers/specs/2026-04-16-shadcn-ui-redesign.md)
> Login ด้วย OWNER account ทุกคน

## วิธีรายงานบัค

ถ้าเจอปัญหา **ถ่าย screenshot** แล้วส่งมาพร้อมบอก:
- หน้าไหน (URL หรือชื่อเมนู)
- คอมหรือมือถือ
- ปัญหาคืออะไร (อ่านไม่ออก / กดไม่ได้ / หน้าพัง / สีแปลก / layout เลื่อน)

---

## กวาง — Layout + ขาย + LIFF (~30 นาที)

### ชุด A: Layout + Navigation (Desktop)

- [ ] เปิดหน้า Login → เห็น logo BESTCHOICE + ปุ่ม**สีเขียว Emerald**
- [ ] Login เข้าระบบ → Sidebar **สีเข้ม (เกือบดำ)** ไม่ใช่น้ำเงิน
- [ ] เมนู active → มีแถบสีเขียวด้านซ้าย + ตัวหนังสือสว่าง
- [ ] คลิกเมนู 3-4 อัน → Breadcrumb บน TopBar เปลี่ยนตามหน้า
- [ ] คลิก "ย่อเมนู" → Sidebar ย่อเป็น icon + hover แสดงชื่อ
- [ ] คลิก "ขยายเมนู" → Sidebar กลับมาปกติ
- [ ] กด ⌘K หรือคลิก search → Command Palette เปิดขึ้น
- [ ] กด Dark Mode (🌙) → ทั้งหน้าเปลี่ยนเป็นสีเข้ม
- [ ] พื้นหลัง content area → **ขาวสะอาด** (ไม่มี tint สีฟ้า)

### ชุด B: Layout + Navigation (มือถือ)

- [ ] เปิดเว็บบนมือถือ → Bottom nav 5 tabs ข้างล่าง ตัว active **สีเขียว**
- [ ] กด ☰ → Sidebar เลื่อนเข้ามาจากซ้าย **สีเข้ม**
- [ ] กดเมนูใน sidebar → ปิด sidebar + เปิดหน้าที่กด

### ชุด C: Flow ขาย (Desktop + มือถือ)

- [ ] เปิดหน้า ลูกค้า → **ไม่มี gradient banner** มีแค่ชื่อหน้า + ปุ่ม
- [ ] ค้นหาลูกค้า → Search ทำงานปกติ
- [ ] คลิกดูรายละเอียดลูกค้า → มีปุ่มกลับ ← + ข้อมูลครบ
- [ ] เปิดหน้า สัญญา → **ไม่มี gradient** + filter tabs ทำงาน
- [ ] คลิกดูรายละเอียดสัญญา → เลขสัญญา + badge สถานะ + ปุ่มกลับ
- [ ] เปิดหน้า POS → ระบบขายทำงานปกติ ปุ่ม**สีเขียว**
- [ ] เปิดหน้า สต็อก → **ไม่มี gradient** + ข้อมูลสินค้าครบ
- [ ] กดปุ่ม "สร้างสัญญา" → ฟอร์มเปิด + input focus **ขอบเขียว**
- [ ] กดยกเลิก → กลับหน้าเดิม

### ชุด D: LIFF (มือถือ)

- [ ] เปิด /liff/contract → **Header สีเขียว Emerald** (ไม่ใช่ gradient น้ำเงิน→เขียว)
- [ ] เปิด /liff/history → ประวัติแสดงปกติ + badge สี
- [ ] เปิด /liff/profile → ข้อมูลโปรไฟล์แสดงครบ

### ชุด E: Flow ขายจริง — เปิดสัญญาครบ loop (Desktop)

- [ ] ไปหน้า ลูกค้า → กด "เพิ่มลูกค้า" → กรอกข้อมูลจริง → บันทึก → ลูกค้าขึ้นในตาราง
- [ ] ไปหน้า POS → ค้นหาลูกค้าที่เพิ่งสร้าง → เลือกสินค้า → เลือก "ผ่อนชำระ" → กดถัดไป
- [ ] กรอกรายละเอียดสัญญา (เงินดาวน์, จำนวนงวด) → ดูตัวอย่างตารางผ่อน → กดสร้างสัญญา
- [ ] สัญญาสร้างสำเร็จ → เปิดดูรายละเอียดสัญญา → ข้อมูลครบ สถานะ Active
- [ ] กดปุ่ม "รับชำระ" ในหน้ารายละเอียดสัญญา → Modal เปิด → กรอกจำนวนเงิน → บันทึก
- [ ] ชำระสำเร็จ → ประวัติชำระขึ้น + จำนวนงวดที่ชำระเพิ่มขึ้น
- [ ] ไปหน้า ใบเสร็จ → เห็นใบเสร็จที่เพิ่งสร้าง → กดดูตัวอย่าง → แสดงผลปกติ

### ชุด F: Flow สต็อก (Desktop)

- [ ] ไปหน้า สต็อก → ดูรายการสินค้า → ค้นหาสินค้าได้
- [ ] กด "เพิ่มสินค้า" → กรอกข้อมูล (ชื่อ, IMEI, ราคาทุน, ราคาขาย) → บันทึก
- [ ] สินค้าขึ้นในรายการสต็อก → สถานะถูกต้อง
- [ ] กดดูรายละเอียดสินค้า → ข้อมูลครบ

### สรุปกวาง

- [ ] **ไม่พบปัญหาเลย** หรือ
- [ ] **พบปัญหา** → ส่ง screenshot + รายละเอียดมา

## ตุ๊กตา — Dashboard + การเงิน + ตั้งค่า (~30 นาที)

### ชุด A: Dashboard (Desktop + มือถือ)

- [ ] เปิด Dashboard → **ไม่มี gradient** มีแค่ชื่อหน้า + ตัวเลข KPI
- [ ] ดู KPI cards → มีเส้นสีข้างซ้าย (เขียว/แดง/เหลือง) + icon + % เปรียบเทียบ
- [ ] ดูกราฟ → กราฟแสดงปกติ สี**เขียว Emerald** เป็นหลัก
- [ ] ดูตารางล่าสุด → Badge เป็นแคปซูลกลม (Active เขียว, Overdue เหลือง)
- [ ] ดูบนมือถือ → Cards ย่อเป็น 2 คอลัมน์ ไม่ล้นจอ

### ชุด B: Flow การเงิน (Desktop)

- [ ] เปิดหน้า ชำระเงิน → **ไม่มี gradient** + ตารางการชำระ
- [ ] กดบันทึกการชำระ → Modal เปิด + overlay สีเข้ม
- [ ] กดยกเลิกใน modal → Modal ปิดปกติ
- [ ] เปิดหน้า ค้างชำระ → **ไม่มี gradient** + badge สีแดง/เหลือง
- [ ] เปิดหน้า คอมมิชชัน → ตารางแสดงปกติ
- [ ] เปิดหน้า ใบเสร็จ → ตาราง + ดูตัวอย่างใบเสร็จได้
- [ ] เปิดหน้า รายงานภาษี → ข้อมูลแสดงครบ
- [ ] Export ข้อมูล (ถ้ามีปุ่ม) → ดาวน์โหลดไฟล์ได้ปกติ

### ชุด C: ตั้งค่า + ระบบ (Desktop)

- [ ] เปิดหน้า ตั้งค่า → **ไม่มี gradient** + tabs ทำงาน
- [ ] เปิดหน้า ผู้ใช้ → ตารางผู้ใช้ + role badge มีสี
- [ ] เปิดหน้า สาขา → ข้อมูลสาขาแสดงครบ
- [ ] เปิดหน้า Audit Logs → ตาราง log แสดงปกติ

### ชุด D: เทสเพิ่มเติม (ถ้ามีเวลา)

- [ ] เปิดหน้า LINE → Broadcast / Rich Menu แสดงปกติ
- [ ] เปิดหน้า Trade-In → ตารางรับซื้อมือสอง
- [ ] เปิดหน้า โปรโมชัน → ข้อมูลโปรโมชัน
- [ ] ลองกดปุ่มลบอะไรสักอย่าง → Confirm dialog เปิดขึ้น overlay สีเข้ม

### ชุด E: Flow การเงินจริง — รับชำระครบ loop (Desktop)

- [ ] ไปหน้า ชำระเงิน → กดบันทึกการชำระ → เลือกสัญญา → กรอกจำนวนเงิน → เลือกวิธีชำระ → บันทึก
- [ ] ชำระสำเร็จ → รายการขึ้นในตาราง + สถานะ Paid
- [ ] ไปหน้า ค้างชำระ → ดูรายการ → badge สถานะถูกต้อง (Overdue สีเหลือง, Default สีแดง)
- [ ] กดเข้าดูสัญญาที่ค้าง → ข้อมูลครบ + ประวัติชำระแสดง
- [ ] ไปหน้า ใบเสร็จ → เห็นใบเสร็จจากการชำระ → กดพิมพ์/ดูตัวอย่าง → แสดงผลปกติ

### ชุด F: Flow ตั้งค่าจริง (Desktop)

- [ ] ไปหน้า ผู้ใช้ → กดเพิ่มผู้ใช้ → กรอกข้อมูล → เลือก role → บันทึก → ผู้ใช้ขึ้นในตาราง
- [ ] แก้ไขข้อมูลผู้ใช้ → บันทึก → ข้อมูลอัปเดต
- [ ] ไปหน้า สาขา → ดูข้อมูลสาขา → แก้ไข → บันทึก → ข้อมูลอัปเดต
- [ ] ไปหน้า ตั้งค่า → เปลี่ยน tab (ทั่วไป/บริษัท/ระบบ) → แต่ละ tab แสดงผลปกติ

### ชุด G: Flow รับซื้อมือสอง (ถ้ามีเวลา)

- [ ] ไปหน้า Trade-In → กดรับซื้อ → กรอกข้อมูลเครื่อง → ตรวจสภาพ → ตีราคา → บันทึก
- [ ] เครื่องมือสองขึ้นในสต็อก → สถานะถูกต้อง

### สรุปตุ๊กตา

- [ ] **ไม่พบปัญหาเลย** หรือ
- [ ] **พบปัญหา** → ส่ง screenshot + รายละเอียดมา

## Visual Checklist (ทั้ง 2 คน)

ดูรวมทั้ง app:
- [ ] **ไม่เห็นสีน้ำเงินเข้ม** (Navy) บน content area, headers, หรือ badges
- [ ] **Sidebar สีเข้ม** (เกือบดำ) ไม่ใช่น้ำเงิน
- [ ] **ปุ่มหลัก** ทุกหน้าเป็น **สีเขียว Emerald**
- [ ] **ไม่มี gradient banner** บนหัวหน้าไหนเลย
- [ ] **Focus ring** เป็นสีเขียวจางๆ เวลาคลิก input/button
- [ ] **พื้นขาวสะอาด** ไม่มี tint ฟ้า/เทาอมฟ้า
- [ ] **Dark mode** ทำงานปกติ ทุกหน้า
